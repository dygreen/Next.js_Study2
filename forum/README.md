## forum
: 게시판 프로젝트

- 상세 페이지 (`Dynamic route`)
- 글 작성/수정/삭제 기능 만들기 (서버)
- 회원 기능 만들기 : `NextAuth` 라이브러리를 이용

  - 아이디/비번 + JWT 사용하기 : `bcrypt`, `Credentials provider` 설정
- 관리자는 모든 글 삭제 가능
- 게시글 댓글 기능 (조회/등록)
